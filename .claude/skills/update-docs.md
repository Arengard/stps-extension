# Update Documentation

Update or create documentation for the STPS extension.

## Steps

1. Ask what documentation is needed:
   - Update function reference in README.md
   - Create usage guide for specific feature
   - Add troubleshooting section
   - Document build process
   - Add examples

2. Review existing documentation:
   - Check README.md
   - Look at howToUseMD/ directory for relevant docs
   - Identify gaps or outdated information

3. Gather information:
   - If documenting functions, read the source code
   - Check test files for usage examples
   - Review git history for recent changes

4. Write/update documentation:
   - Use clear, concise language
   - Include code examples
   - Add expected output
   - Mention common pitfalls
   - Follow existing doc structure

5. Verify accuracy:
   - Test any code examples
   - Check function signatures match code
   - Ensure installation/build steps are current

6. Organize documentation:
   - Place in appropriate location
   - Update table of contents if needed
   - Link from README.md if new doc

## Context

Documentation is in README.md and howToUseMD/ directory. The extension provides specialized functions for German tax office data processing.
