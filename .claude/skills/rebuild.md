# Rebuild Extension

Quickly rebuild the STPS DuckDB extension using the quick-rebuild.bat script.

## Steps

1. Run the quick-rebuild.bat script
2. Check for any compilation errors
3. Report build status and any issues found
4. If successful, confirm the extension binary location

## Context

This project uses a Windows build system with CMake and MSVC. The quick-rebuild script is the fastest way to rebuild after code changes.
